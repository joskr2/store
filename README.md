# Store clone

Store clone is a ecommerce inspired on Amazon.

## Installation

Clone this repository on your local machine 



## Usage

```javascript
npm install

npm start 
```

## Tech stack
Frontend: 

1. React
2. Saas

Backend:

1. Firebase(auth)
2. Firestore(store users and orders)
3. Stripe(payment method)
